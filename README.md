# NadimRamrattan.CEN3024-31032assignment3
SDLC Assignment part 2
Library Management system 
The purpose of this software development plan is to create a system that is intended to help a librarian manage patron records through a simple console-based Java application. The program will allow the librarian to store patron information, import patron records from a file, add new patrons manually, remove patrons when necessary, and view all patron records currently stored in the system.
